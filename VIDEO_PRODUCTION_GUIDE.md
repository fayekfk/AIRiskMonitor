## üé• Production Checklist

### Pre-Production

- [ ] **Prepare Sample Data**
  - Use `sample_project_data.csv` or `sample_project_advanced.csv`
  - Ensure data shows variety of risk levels (high, medium, low)

- [ ] **Clean Browser Environment**
  - Clear browser cache and localStorage
  - Use incognito/private mode for clean recording
  - Close unnecessary tabs and notifications

- [ ] **Test Application**
  - Run `npm run dev` to start development server
  - Test all features you'll demonstrate
  - Ensure AI API is working (OpenAI key is valid)
  - Test EmailJS integration

- [ ] **Prepare Recording Setup**
  - Install screen recording software (see recommendations below)
  - Test microphone quality
  - Set screen resolution to 1920x1080 (Full HD)
  - Close distracting applications

### Recording Tools (Recommended)

**Free Options:**
1. **OBS Studio** (Windows/Mac/Linux) - Professional, free, open-source
2. **Loom** (Web-based) - Easy to use, free tier available
3. **ShareX** (Windows) - Free, lightweight
4. **QuickTime** (Mac) - Built-in screen recording
5. **Windows Game Bar** (Windows 10/11) - Built-in, press Win+G

**Paid Options:**
1. **Camtasia** - Professional editing features
2. **ScreenFlow** (Mac) - Excellent for tutorials
3. **Snagit** - Simple and effective

### Recording Settings

- **Resolution:** 1920x1080 (1080p)
- **Frame Rate:** 30 FPS minimum, 60 FPS preferred
- **Audio:** 48kHz, 16-bit minimum
- **Format:** MP4 (H.264 codec)
- **Bitrate:** 5-10 Mbps for high quality

### During Recording

- [ ] **Speak Clearly**
  - Pace yourself - not too fast
  - Pause between sections
  - Smile while speaking (it affects your tone!)

- [ ] **Mouse Movements**
  - Move cursor slowly and deliberately
  - Highlight important elements
  - Use cursor highlighting if available

- [ ] **Timing**
  - Follow the script timing
  - Allow 2-3 seconds for viewers to read on-screen text
  - Don't rush through features

### Post-Production

- [ ] **Edit Video**
  - Remove mistakes and long pauses
  - Add transitions between scenes
  - Include zoom-ins for important details
  - Add background music (optional, keep it subtle)

- [ ] **Add Enhancements**
  - On-screen text overlays (as per script)
  - Highlight cursor clicks
  - Add callout boxes for key features
  - Include intro/outro graphics

- [ ] **Add Captions/Subtitles**
  - Auto-generate with YouTube or editing software
  - Review and correct errors
  - Helps with accessibility and SEO

### Export Settings

- **Format:** MP4
- **Codec:** H.264
- **Resolution:** 1920x1080
- **Frame Rate:** 30 FPS
- **Audio:** AAC, 192 kbps
- **File Size:** Aim for under 500MB for easy sharing

## üé§ Voiceover Tips

### Tone & Delivery
- **Professional but friendly** - You're a helpful expert, not a robot
- **Enthusiastic** - Show excitement about the features
- **Clear enunciation** - Speak each word clearly
- **Moderate pace** - 140-160 words per minute
- **Natural pauses** - Breathe between sentences

### Recording Environment
- **Quiet room** - No background noise
- **Microphone distance** - 6-8 inches from mouth
- **Pop filter** - Reduces plosive sounds (P, B, T)
- **Room treatment** - Record in carpeted room or use blankets to reduce echo

### Script Reading
- **Practice first** - Read script 3-4 times before recording
- **Mark pauses** - Note where to pause in script
- **Record in sections** - Don't try to do it all in one take
- **Stay hydrated** - Keep water nearby

## üé¨ Quick Start Recording Guide

### Option 1: Simple Loom Recording (Fastest)

1. Go to [loom.com](https://www.loom.com)
2. Install browser extension
3. Click Loom icon ‚Üí "Screen + Camera" or "Screen Only"
4. Select browser tab with your app
5. Click "Start Recording"
6. Follow the script
7. Click "Stop" when done
8. Loom auto-uploads and provides shareable link

**Pros:** Super easy, instant sharing, built-in editing
**Cons:** Limited editing features, watermark on free plan

### Option 2: OBS Studio (Professional)

1. Download [OBS Studio](https://obsproject.com/)
2. Create new scene
3. Add "Display Capture" or "Window Capture" source
4. Add "Audio Input Capture" for microphone
5. Click "Start Recording"
6. Follow the script
7. Click "Stop Recording"
8. Edit in video editor (DaVinci Resolve, Shotcut - both free)

**Pros:** Professional quality, no watermarks, full control
**Cons:** Steeper learning curve, requires editing software
